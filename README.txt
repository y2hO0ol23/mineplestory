1.무기 옵션 추가
도구 하나당 3개의 옵션이 붙습니다.

1.1 등급
등급은 노멀, 레어, 에픽, 유니크, 레전더리 총 5개로 나뉘어져 있습니다.
만들어진 종류에 따라 등급이 나뉘어집니다.
나무, 가죽, 활, 석궁 : 노멀
돌, 사슬, 금 : 레어
철 : 에픽
다이아 : 유니크
삼지창, 네더라이트, 거북모자 : 레전더리
등급은 큐브를 사용하여 확률적으로 올릴 수 있습니다.

1.2 능력치
채력, 방어력, 넉백방어, 이동속도, 점프력, 데미지, 공격속도, 치명타 총 8가지입니다.
첫번째 줄의 능력치는 도구의 등급과 같은 등급이 되며, 나머지 두 줄은 확률적으로 같은 등급이 되거나, 하나 낮은 등급이 됩니다.
치명타의 경우 근접 무기들은 철골렘이 파괴되는 소리가 들리며, 활의 경우 크리티컬 파티클이 생기게 됩니다.

1.3 큐브
발전과제 달성시 task종류는 노멀, goal종류는 레어, challenge종류는 에픽등급의 큐브를 5개 지급합니다.
큐브 등급에 따라 아이템의 등급이 높아질 확률과 같은 등급의 능력치가 될 확률을 높일 수 있습니다.
이름표 1개, 자수정 조각 4개, 우는 흑요석 4개를 사용하여 큐브 하나를 제작할 수 있습니다. 또한 고대잔해 1개를 사용하여 하나의 큐브 등급을 높일 수 있습니다.
사용방법
도구 능력치 재배치 : 큐브를 왼손에, 해당 도구를 오른손에 들어준 뒤 손을 바꾸어줍니다.
큐브 등급 강화 : 큐브를 오른손에, 고대잔해를 왼손에 들어준 뒤 손을 바꾸어줍니다.
이때 더 적은 양의 아이템을 기준으로 사용되니 주의해주세요

1.4 스타포스
스타포스를 사용하여 추가옵션을 더욱 강력하게 만들 수 있습니다. 1성부터 15성까지 5%, 16성부터 20성까지 15%의 능력치를 추가로 얻을 수 있게 됩니다.
에메랄드를 사용하여 스타포스 강화를 합니다. 에메랄드를 왼손에, 해당 도구를 오른손에 들어준 뒤 손을 바꾸어줍니다.

1.5 인첸트 강화
모루를 왼손에 들고 손을 바꾸면 경험치를 소모하며 랜덤한 하나의 인첸트를 강화합니다. 25%확률로 새로운 옵션이 붙습니다.

2. 무기들의 특수효과
방패 : 공격을 타이밍에 맞춰 막으면 방패를 들고있는 짧은 시간동안 이동속도가 증가합니다.
석궁 : 다중발사가 최대 10까지 적용됩니다. 더 많은 화살들이 날아가긴 하다만... 랙을 조심하세요!
활 : 힘 효과가 있을때 더 멀리 날아가고, 나약함 효과가 있을때 더 적게 날아갑니다. 활을 쏠때도 포션 효과가 있으면 좋겠죠?

3. 엔더월드
이제 5개의 발전과제를 성공하지 못하면 엔더진주를 사용할 수 없습니다.
삼림대저택, 네더 요세, 보루 잔해의 몹들을 전부 죽여야 하며, 위더, 엘더가디언을 잡아야합니다.
다만 월드 자체의 발전과제이기에 친구가 많다면 금방 갈 수 있겠죠?
삼림대저택 : 변명자, 소환사
네더 요세 : 블레이즈, 스켈레톤, 좀비피글린
보루 잔해 : 피글린 야수, 피글린, 호글린
모든 도전과제를 클리어 하려면 cartoonish:the_end/clear 함수를 사용해주세요

4. 몬스터
몬스터들이 스폰 지점에서 멀어질수록 더욱 강력해집니다.
또한 보스몹이 강화가 됩니다. 엔더드래곤, 위더, 엘더가디언의 채력이 512로 바뀝니다.
가디언, 레이드에 출현하는 몬스터들은 8레벨로 고정됩니다

4.1 위더
위더는 더욱 강력한 몬스터가 되었습니다. 시간마다 데미지를 받을경우 위더 스켈레톤 3마리를 소환합니다.
채력이 반 아래로 내려가게 되면 위더는 데미지를 받을 때마다 폭발합니다.
